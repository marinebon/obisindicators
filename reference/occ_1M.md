# OBIS occurrences, global sample of 1 million records

Simple dataset of species and location for example purposes from
2022-04-04.

## Usage

``` r
occ_1M
```

## Format

A data frame with 1,000,000 rows and 4 variables:

- `decimalLongitude`: unique x

- `decimalLatitude`: unique y

- `species`: unique species

- `records`: number of records based on unique values of x, y, species

<https://obis.org/data/access>
